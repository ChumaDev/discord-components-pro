## [0.1.0] - 2026-02-22

### Added
- Complete Discord Components v2 SDK with React 18 and TypeScript 5.9
- 16 components: Button, TextDisplay, StringSelect, Container, ActionRow, Separator, Dialog, EmojiPicker, ColorPicker, FileUpload, MediaGallery, MarkdownPreview, Thumbnail, SortableItem, Icon, MessageBuilder
- Full drag & drop support with @dnd-kit for component reordering
- Undo/Redo system with history tracking in Zustand store
- Keyboard shortcuts (Ctrl+Z for undo, Ctrl+Y for redo)
- Export to 4 formats: JSON, Discord.js, TypeScript, Python
- Import functionality supporting JSON with validation
- Template system with 4 pre-built templates
- Preset library with 9 configurations across 5 categories
- Performance optimization utilities (debounce, throttle, lazy loading)
- 13 feature modules with professional architecture
- Component search and filtering system
- Batch operations (delete, update, duplicate, move, sort)
- Analytics tracking with event logging
- Custom theme system (Dark/Light)
- Markdown parsing with Discord syntax support
- Clipboard operations with ID regeneration
- Comprehensive validation with Zod schemas
- 305 unit tests with 85% coverage
- Storybook with stories for all components
- Complete documentation

### Testing
